# GradeBook Analyzer

This project is a Python program that helps analyze student marks. It can take input manually or from a CSV file and then calculates statistics and assigns grades.

## Features

* Enter student marks manually
* Load marks from a CSV file
* Calculate average, median, highest, and lowest marks
* Assign grades (A, B, C, D, F)
* Show passed and failed students
* Display results in a table

## CSV Format

The CSV file should look like this:

```
Name,Marks
Aanya,85
Riya,90
Kabir,72
```

## How to Run

1. Save the Python file and CSV file in the same folder.
2. Open terminal or PowerShell.
3. Run the program:

```
python gradebook.py
```

4. Choose an option from the menu.

## Requirements

* Python 
* Uses only the built-in `csv` module
